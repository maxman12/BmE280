# BmE280 ESP32 S2 Mini Project

Starter project for ESP32 S2 Mini and BME280 sensor.

Please refer to [PROJECT_INFO.md](./PROJECT_INFO.md) for detailed technical specifications, tech stack, and roadmap.

## Getting Started
1. Install [PlatformIO](https://platformio.org/).
2. Open this folder in your IDE.
3. Build and upload to your ESP32 S2 Mini.
